# Healync — Backend Server

REST API server for the [Healync](https://healync.vercel.app) doctor appointment platform.

🔗 **Frontend:** [github.com/tawchifulislam/healnyc](https://github.com/tawchifulislam/healnyc)

---

## Tech Stack

- **Node.js + Express.js** — REST API
- **MongoDB** — Database
- **Vercel** — Deployment

---

## Getting Started

```bash
git clone https://github.com/tawchifulislam/healnyc-server.git
cd healnyc-server
npm install
```

Create a `.env` file:

```env
MONGODB_URI=your_mongodb_connection_string
PORT=5000
CLIENT_URL=http://localhost:3000
```

```bash
node index.js
```
